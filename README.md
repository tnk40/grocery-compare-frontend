# UK Grocery Compare

Compare grocery prices across UK supermarkets to find the cheapest place to shop.

## Live Site

https://tnk40.github.io/grocery-compare-frontend/

## Features

- Build shopping lists
- Compare prices across multiple UK supermarkets to make more informed shopping decisions
- See which store is cheapest for your basket
- Save and load lists with a free account

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (GitHub Pages)
- **Backend**: Python FastAPI (Railway)
- **Database**: PostgreSQL
